You Need a Web Budget

Hi! I built this YNAB demo site specifically to send with my resume to the YNAB folks so don't plan on seeing this site stick around. It reads YNAB budgets from Dropbox accounts and gives a web interface into them. It's just a viewer for budgets. It's not feature complete like the desktop client, but it's still a neat demonstration.

Keep in mind I researched the YNAB data format and wrote this in less than two weeks after hours so expect to see some bugs.

Without further ado, click the links below to log in with your Dropbox account or the Github button to view the full source.
